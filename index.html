ThmdPlayer

Lekki, szybki odtwarzacz wideo dla Linux. Zbudowany w C# / .NET 8, AvaloniaUI i LibVLC.

Pobierz Zobacz na GitHub
Linux .NET 8 AvaloniaUI LibVLC

Funkcje

Odtwarzanie wideo

Obsluga popularnych formatow: MP4, MKV, AVI, MOV, WMV, FLV, WebM dzieki silnikowi LibVLC.

Playlista & kolejka

Zapis i przywracanie listy odtwarzania. Kolejka odtwarzania, drag & drop, wyszukiwanie i sortowanie.

Pomijanie intra / outra

Reczne ustawienia oraz auto-detekcja przez ffmpeg (blackdetect, silencedetect, scenedetect).

Skroty klawiszowe

Spacja (play/pause), strzalki (nawigacja / glosnosc), F (fullscreen), N / P (nastepny / poprzedni).

Napisy & synchronizacja

Automatyczne ladowanie .srt, synchronizacja +/- 0.5s, wybor sciezki audio.

Tryb Mini Player

Zmniejsz okno do 400x280, zawsze na wierzchu, tryb pelnoekranowy.

Pasek sterowania

Oddzielne okno kontrolne z suwakiem postepu, czasem, glosnoscia i predkoscia odtwarzania.

Wznawianie pozycji

Kazdy plik zapamietuje miejsce zatrzymania i wznawia odtwarzanie od tego momentu.

Instalacja

Wymagania

  • .NET 8.0 SDK
  • Linux (Ubuntu, Debian, Fedora, Arch...)
  • VLC Media Player z bibliotekami libvlc-dev

Szybki start (Ubuntu / Debian)

sudo apt-get update
sudo apt-get install -y vlc libvlc-dev
wget https://packages.microsoft.com/config/ubuntu/22.04/packages-microsoft-prod.deb -O packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
sudo apt-get update
sudo apt-get install -y dotnet-sdk-8.0

Budowanie ze zrodel

git clone https://github.com/softbery-org/Thmd-Linux.git
cd ThmdPlayer
dotnet restore
dotnet build ThmdPlayer.csproj
dotnet run --project ThmdPlayer.csproj
